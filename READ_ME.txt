Это проект с технологиям laravel + vue.js.
Привлекаются сторонние библиотеки - phpmailer, jquery, fancybox, axios и др.
Используется препроцессор SASS и сборщик модулей js - Vite.
Идёт взаимодействие со сторонними API - Wazzup, google/apiclient (интеграция с гугл таблицами), Bitrix24 REST API
Инфомация о поставщиках и технике подгружается из гугл таблицы раз в день с помощью Cron.
Каждый зарегистрированный пользователь имеет свой уникальный номер телефона (т.е. нельзя зарегистрироваться например и заказчиком и поставщиком под одним номером телефона).

Дамп базы данных - base.sql

Файлы инструкция для пользователей, инструкция для разработчиков,тех-задание и пример гугл таблицы лежат в папке Docs.
Так же в приложении есть панель администратора. Её функционал можно посмотреть в инструкции для пользователей.

Ссылка ---- https://examplesites.tw1.ru/ (здесь можете потестить)
Это версия для тестов, на продакшине например анкета менеджера при регистрации попадает на модерацию к администратору. Пока анкета не будет подтверждена, менеджер не сможет зайти в свой кабинет (ссылка на продакшн - https://sbm-form.tw1.ru/)
